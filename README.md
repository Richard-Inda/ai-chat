# AI Chat - Coding Test Submission

This repository is an overview and submission container for the AI chat coding challenge.

Live app: [www.RichardAi.chat](https://www.RichardAi.chat)

## Project Context

- This is the coding test submission repository.
- The implementation is split across two dedicated repositories:
  - Frontend repository (`chat-ai-ui`)
  - Backend repository (`chat-ai-api`)
- The deployed experience is available at [www.RichardAi.chat](https://www.RichardAi.chat).

## What This App Does

- Accepts a user prompt and submits it to an AI-enabled backend.
- Fetches and displays AI responses dynamically in the chat UI.
- Includes loading and error states for request handling.
- Supports chat history and clear-history behavior.

## Assessment Overview

- **Role:** Frontend Engineer
- **Challenge:** Build an AI-integrated web app with clean UI and solid architecture
- **Stack:** React, Angular, Vue or NextJS (your choice)
- **Deadline:** 7 days from assignment date

### Objective

Create a lightweight app where a user can input a prompt, submit it to an AI API (for example OpenAI), and receive/display a response. Focus areas:

- Prompt input + submit button
- Fetching from OpenAI or HuggingFace
- Displaying results dynamically
- Error handling and loading states

### Bonus (+10 points)

- Save and show past prompts/responses (chat history)
- Include a "Clear" button for the user

### Submission Instructions

- Create a public GitHub repository
- Include a simple `README.md` with setup instructions
- Start immediately; no need to wait for a live interview

## Tech Snapshot

- Frontend: Vue 3 + TypeScript + Vite + Pinia + Tailwind
- Backend: Node.js + Express + TypeScript
- AI + Data: OpenAI, Stream Chat, Neon Postgres, Drizzle ORM
